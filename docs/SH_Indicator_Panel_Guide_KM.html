RSI ខាងលើ 50

Tool បង្ហាញបារពណ៌ Green។ Momentum ទូទៅលំអៀងទៅ Bullish។ ចន្លោះ 55–70 ជាតំបន់ Bullish ដែលមានប្រយោជន៍ជាងការមើលត្រឹម 70។

RSI ខាងក្រោម 50

Tool បង្ហាញបារពណ៌ Blue។ Momentum ទូទៅលំអៀងទៅ Bearish។ ចន្លោះ 30–45 ជាតំបន់ Bearish context។

RISING / FALLING

RISING = RSI បច្ចុប្បន្នខ្ពស់ជាង bar មុន។ FALLING = RSI បច្ចុប្បន្នទាបជាង bar មុន។ Direction នេះអាចសំខាន់ជាងលេខតែមួយ។

តំបន់ 45–55

ជាតំបន់ Neutral/Mixed។ ជាទូទៅគួររង់ចាំ Price ចេញពី range ឬ Diamond Zone មាន closed-candle reaction ច្បាស់។

កុំច្រឡំ៖ RSI ≥ 70 មិនមែន Sell ដោយស្វ័យប្រវត្តិទេ។ Trend ខ្លាំងអាចនៅ Overbought បានយូរ។ RSI ≤ 30 ក៏មិនមែន Buy ដោយស្វ័យប្រវត្តិដែរ។

MACD + RSI + Market Pressure

Context only

ALIGNED BEARISH

MACD < 0

MIXED

មិនស្របគ្នា

Market Pressure ប្រាប់អ្វី?

Tool ផ្សំសញ្ញា MACD 20 bars ចុងក្រោយ និងចម្ងាយ RSI ពី 50 ទៅជា Bullish / Bearish / Neutral។ វាជា Context score មិនមែន win-rate ឬ probability ទេ។
