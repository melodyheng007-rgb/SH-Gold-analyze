15M Context -> 5M Confirmation

  • មើល Location និង Diamond Zone នៅ 15M។
  • រង់ចាំ 5M closed candle បង្ហាញ rejection/reclaim។
  • កុំសម្រេចចិត្តពី Candle ដែលមិនទាន់បិទ។

សម្រាប់ Swing

  • មើល Direction និង Range នៅ 4H។
  • រង់ចាំ 1H reaction នៅ Diamond Zone។

04 / DIAMOND LINE

របៀបមើល DL SUP និង DL RES

DL SUP / BUY KEY ZONE

Support context

  • ល្អ: Price test zone ហើយ closed candle reclaim/hold លើ zone។
  • រង់ចាំ: Status `TESTED` ប៉ុន្តែ `MTF Agreement` មិនទាន់បាន។
  • ខ្សោយ: Candle បិទក្រោម zone ជាបន្តបន្ទាប់ ឬ structure ខូច។
DL RES / SELL KEY ZONE

Resistance context

  • ល្អ: Price test zone ហើយ closed candle reject/hold ក្រោម zone។
  • រង់ចាំ: Zone ថ្មី ប៉ុន្តែ confirmation timeframe មិនទាន់ស្រប។
  • ខ្សោយ: Candle បិទលើ zone ជាបន្តបន្ទាប់ ឬ structure ត្រូវបាន reclaim។

07 / QUICK REFERENCE

Checklist មុនប្រើ Key Levels

ត្រូវពិនិត្យតាមលំដាប់

  1. Market ត្រឹមត្រូវ: XAU ឬ BTC។
  2. Position ត្រឹមត្រូវ: Scalp ឬ Swing។
  3. FEED MATCHED និង OHLC CLEAN។
  4. PRICE នៅណាធៀប OP និង MLP។
  5. PRICE នៅជំហាន K ណា។
  6. DL SUP ឬ DL RES នៅជិត Price ឬអត់។
  7. State: FRESH, TESTED, CONFIRMED ឬ INVALIDATED។
  8. រង់ចាំ closed candle និង MTF Agreement។

មិនគួរធ្វើ

  • Buy ព្រោះប៉ះ K-3 តែមួយមុខ។
  • Sell ព្រោះប៉ះ K+3 តែមួយមុខ។
  • ចូលតាម Diamond ដែលមាន State = TESTED ប៉ុណ្ណោះ។
  • គិតថា Grade B 75 = ឈ្នះ 75%។
  • ប្រើ Level ពេល DATA LOCK ឬ UNMATCHED FEED។
  • Chase Price ដែលរត់ឆ្ងាយពី Diamond Zone។
  • សម្រេចពី Candle ដែលកំពុង forming។
